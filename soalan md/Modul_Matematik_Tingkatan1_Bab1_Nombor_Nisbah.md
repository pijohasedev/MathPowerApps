### S30.
Diberi suhu pada pukul 11:00 a.m. = 30°C

(a) Pukul 7:00 p.m. → turun 4°C → 5:00 a.m. → naik 8°C → 11:00 a.m.
    Suhu pada 11:00 a.m. = Suhu pada 7:00 p.m. – 4 + 8 = 30
    Suhu pada 7:00 p.m. = 30 – 8 + 4 = **26°C**

(b) Pukul 11:00 a.m. = 30°C
    Tiga jam kemudian (pukul 2:00 p.m.) → naik 2°C
    Suhu pada pukul 2:00 p.m. = 30 + 2 = **32°C**

### S49.
Jumlah soalan = 30
Soalan dijawab = 25
Soalan tidak dijawab = 5
Markah untuk tidak dijawab = 5 × (–1) = –5

Katakan bilangan betul = x
Bilangan salah = 25 – x

Jumlah markah = 5x + (–3)(25 – x) + (–5) = 59
               5x – 75 + 3x – 5 = 59
               8x – 80 = 59
               8x = 139
               x = 17.375

Semak: Bilangan betul mestilah integer. Mari semak semua kemungkinan:
Jika betul = 17, salah = 8, tidak jawab = 5
Markah = 17(5) + 8(–3) + 5(–1) = 85 – 24 – 5 = 56 ≠ 59

Jika betul = 18, salah = 7, tidak jawab = 5
Markah = 18(5) + 7(–3) + 5(–1) = 90 – 21 – 5 = 64 ≠ 59

*Semakan semula:*
Markah salah dan tidak jawab digabung:
Jumlah soalan dijawab = 25
Tidak dijawab = 5

Katakan betul = x, salah = 25 – x
5x + (–3)(25 – x) + 5(–1) = 59
5x – 75 + 3x – 5 = 59
8x – 80 = 59
8x = 139

**Tiada penyelesaian integer.** Mari cuba tafsiran lain:
Peserta menjawab 25 soalan, 5 tidak dijawab.
Jumlah markah = 59.

Jika betul = 15, salah = 10, tidak jawab = 5:
15(5) + 10(–3) + 5(–1) = 75 – 30 – 5 = 40

Jika betul = 17, salah = 8, tidak jawab = 5:
17(5) + 8(–3) + 5(–1) = 85 – 24 – 5 = 56

Jika betul = 18, salah = 7, tidak jawab = 5:
18(5) + 7(–3) + 5(–1) = 90 – 21 – 5 = 64

Jawapan yang paling munasabah: antara 56 dan 64, 59 tidak tercapai dengan tepat.
*Mungkin maksud soalan berbeza.* Semak semula:

Jika semua 30 soalan dijawab (25 dijawab + 5 tidak dijawab sebenarnya bermakna 5 tidak dijawab):

5x + (–3)(25 – x) + 5(–1) = 59
5x – 75 + 3x – 5 = 59
8x = 139
x = 17.375

Oleh kerana bilangan soalan mestilah integer, maka:
Bilangan betul = **17**, salah = 8, tidak jawab = 5, markah = 56
Bilangan betul = **18**, salah = 7, tidak jawab = 5, markah = 64

Tiada jawapan tepat 59 dengan data ini. Mungkin terdapat ralat dalam data soalan atau markah tidak dijawab berbeza. Untuk tujuan latihan, jawapan yang paling hampir:
**17 soalan betul** (menghampiri 59)
